Data Cleaning
Bersihkan dan perbaiki data sebelum analisis

Menangani Missing Values —

Menghapus baris/kolom dengan terlalu banyak data kosong, atau mengisi nilai yang hilang menggunakan rata-rata, median, modus, atau nilai tetap sesuai kebutuhan data.

Menghapus Data Duplikat —

Mencari dan menghapus data yang tercatat lebih dari satu kali. Duplikat dapat terjadi saat data dikumpulkan dari beberapa sumber atau proses input yang berulang.

Memperbaiki Inkonsistensi Data —

Menyamakan format penulisan pada kolom kategorikal. Contoh: "Laki-laki", "laki laki", dan "L" diubah menjadi satu format yang sama untuk konsistensi analisis.

Menangani Outlier —

Mengidentifikasi nilai yang sangat berbeda dari data lainnya. Tentukan apakah outlier merupakan kesalahan input atau nilai yang valid sebelum memutuskan untuk menghapusnya.

Mengubah Tipe dan Format Data —

Mengubah data teks menjadi numerik jika diperlukan dan menyamakan format tanggal (misalnya dari "10/06/2026" menjadi "2026-06-10" format standar ISO 8601).

Validasi Hasil Pembersihan Belum divalidasi

Memastikan tidak ada kesalahan baru setelah proses cleaning. Mengecek kembali konsistensi, kelengkapan, dan kualitas data yang telah dibersihkan.
